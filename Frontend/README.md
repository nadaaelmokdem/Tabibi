# Tabibi Frontend (Vite + React + TypeScript + Tailwind CSS v4)

This folder contains the client-side SPA (Single Page Application) for the Tabibi platform.

For a comprehensive explanation of the entire project, architecture, key workflows, database schema, payment gateways, and step-by-step guides, please refer to the **main project README** at the root of the workspace:

👉 **[Root README.md](../README.md)**

## 🏃 Quick Start (Frontend Only)

1. Ensure you have Node.js (v18+) and npm installed.
2. Configure your environment variables in `.env`:
   ```env
   VITE_API_BASE_URL=https://tabibi.dpdns.org/api
   VITE_GOOGLE_CLIENT_ID=YOUR_GOOGLE_CLIENT_ID
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:5173](http://localhost:5173) in your browser.
